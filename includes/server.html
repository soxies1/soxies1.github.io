Raspberry Pi Linux Server

Running Apache2, Tyson's home server has proved to be his most useful project to-date. Hosting this and various other websites for friends, the server has been configured to allow SSH login, and seperate user accounts for friends to explore the server and manage their hosted content.

Served Content Using LAMP Stack

LAMP was the first framework Tyson used to host a web-app, which was a Random Question Survey application (details in "Websites" section). Setting up the LAMP framework involved installing and configuring MySQL, Apache2, and PHP.

Server Content Using MEAN Stack

Tyson has also experimented with the MEAN stack a few times as well. Two application he implemented with the framework were: Study buddy, and YouHaveGotToBeKitten.Me (details on both in the "Website" section).

Puppet and Vagrant Deployment

After one of Tyson's Co-op placements involved working closely with Puppet, he applied his acquired knowledge to his home server. He used Puppet to manage his Linux server and various Vagrant instances, which would be used as either external clients or private LAN repositories.
